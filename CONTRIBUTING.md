# Contributing to SOaC-Enterprise

Thank you for contributing to the **SOaC** standard. 

### How to Contribute
1. Fork the [SOaC-Enterprise](https://github.com/ge0mant1s/SOaC-Enterprise) repo.
2. Create a branch for your artifact (e.g., `feat/detection-okta-mfa`).
3. Ensure your artifact includes the **SOaC Metadata Header**.
4. Submit a Pull Request.
